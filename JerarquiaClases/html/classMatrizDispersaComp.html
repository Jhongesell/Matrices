DescripciÃ³n detallada

Clase para el trabajar con matrices dispersas de punto flotante basada en el algoritmo Jagged Diagonal Storage (JDS) que puede ser compactada, pero soporta matrices con todos los valores distintos. El algoritmo esta optimizado para hacer el producto matriz vector. El algoritmo de compactación presupone que se llenara la matriz renglon por renglon de manara consecutiva, no asi las columnas dentro del renglon. El algoritmo de compactación sólo guarda los renglones distintos de la matriz. Si la matriz tiene ciertas propiedades es posible usar otros algoritmo de compactación (matrices tridiagonales con coeficientes constantes solo guardar los 3 valores distintos)
